# Webpack start
> A project to development JS application from ES standards

### Version
1.0.0

## Usage

### Installation dependencies

```sh
$ npm install
```

#### Recommended versions
* NodeJS 11.9.0
* Npm 6.5.0

### Getting Started
Development

```sh
$ npm run dev
```

Production

```sh
$ npm run build
```
